# Python String Manipulation

This module covers string manipulation in Python, part of [Corey Schafer's Python Programming Beginner Tutorials](https://www.youtube.com/playlist?list=PL-osiE80TeTskrapNbzXhwoFUiLCjGgY7).

## Files Overview

1. `strings.py` - [Video: Strings - Working with Textual Data](https://youtu.be/k9TUPpGqYTo)
   - String basics
   - String creation methods
   - Escape characters
   - Multi-line strings

2. `string_methods.py` - [Video: String Methods](https://youtu.be/k9TUPpGqYTo?t=431)
   - Common string methods
   - Case manipulation
   - String searching and modification
   - String validation methods

3. `string_slicing.py` - [Video: String Slicing](https://youtu.be/ajrtAuDg3yw)
   - Basic slicing syntax
   - Positive and negative indices
   - Step values in slicing
   - Practical applications

4. `string_concatenation.py` - [Video: String Formatting](https://youtu.be/vTX3IwquFkc)
   - String concatenation methods
   - String formatting techniques
   - f-strings (Python 3.6+)
   - Template strings

5. `string_method_example.py`
   - Practical examples combining various string operations
   - Real-world use cases
   - Best practices

## Key Concepts

- String Immutability
- String Methods and Operations
- String Formatting
- String Slicing and Indexing
- Escape Characters
- String Concatenation

## Usage

Each file can be run independently:
```bash
python strings.py
python string_methods.py
# etc.
```
